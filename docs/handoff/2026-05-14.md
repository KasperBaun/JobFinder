  posting; if present, prefer it over the keyword scan.

silent — preserves current behaviour for the long tail.

### Wait-loop gotcha (small, save yourself the headache)

If you stand up the SSE smoke the way I did, write to a fresh log path
per run (eg `sse-$(date +%s).log`) instead of reusing the same file
between smokes. The bash wait loop's first iteration can race with a
stale file from a previous run and exit prematurely. Reading the
notification "task completed" without checking how far the search
actually got cost me one wasted run.
