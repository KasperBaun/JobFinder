Run the `jobmatch` CLI `skillset` subcommand to author or refresh `config/skillset.md`.

From the repo root, execute:

```
dotnet run --project src/Jobmatch.Cli -- skillset $ARGUMENTS
```

Phase 1 stub — the interactive prompt flow lands in Phase 2, richer prompts in Phase 5.
